Quick API endpoints

These are REST endpoints (not web pages):

  • GET /jobs/search — search jobs
  • GET /notes — list saved notes
  • POST /notes — create a note
  • PUT /notes/{id} — update a note
  • DELETE /notes/{id} — delete a note
